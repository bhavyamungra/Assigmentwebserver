Step 1: Update Package Index

First, ensure that the local package index is up-to-date by running:

sudo apt update

Step 2: Install MySQL Server

Use the following command to install the MySQL server:

sudo apt install mysql-server

Step 3: Secure MySQL Installation

After installation, run the security script that comes with MySQL to secure the installation:

sudo mysql_secure_installation

Follow the on-screen prompts to configure the security options for your MySQL installation.

Step 4: Verify MySQL Installation

After securing the installation, you can verify that MySQL is running correctly by checking its status:

sudo systemctl status mysql

This command should display the status of the MySQL service, indicating whether it is active or not.
