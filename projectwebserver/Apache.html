Step 1: Update Package Index

First, ensure that the local package index is up-to-date by running:

sudo apt update

Step 2: Install Apache

Use the following command to install Apache:

sudo apt install apache2

Step 3: Adjust Firewall (if applicable)

If a firewall is enabled, ensure that it allows connections to Apache. You can enable the ufw firewall to allow Apache connections by typing:

sudo ufw allow 'Apache'

Step 4: Verify Apache Installation

After installation, Apache should already be running. You can verify this by entering your server's public IP address in your web browser. If Apache is running, you should see the default Apache landing page.

Step 5: Enable Apache to Start on Boot (Optional)

If you want Apache to start automatically every time your server boots, run the following command:

sudo systemctl enable apache2

Step 6: Restart Apache (Optional)

If you made any changes to the Apache configuration, you can restart Apache to apply the changes by typing:

sudo systemctl restart apache2
