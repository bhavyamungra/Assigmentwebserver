Step 1: Log in to MySQL

Log in to the MySQL server as the root user:

mysql -u root -p

Enter the root password when prompted.

Step 2: Create a New Database

Use the following SQL query to create a new MySQL database:

CREATE DATABASE dbname;

Replace 'dbname' with the desired name of the new database.

Step 3: Grant User Permission to the Database

Grant appropriate privileges to the user for the new database. For example, to grant all privileges on the new database to the user:

GRANT ALL PRIVILEGES ON dbname.* TO 'username'@'localhost';

Replace 'dbname' with the name of the new database and 'username' with the username of the user.

Step 4: Flush Privileges

Flush the privileges to apply the changes:

FLUSH PRIVILEGES;

Step 5: Exit MySQL

Exit the MySQL prompt:

exit;
