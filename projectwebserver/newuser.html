Step 1: Log in to MySQL

Log in to the MySQL server as the root user:

mysql -u root -p

Enter the root password when prompted.

Step 2: Create a New MySQL User

Use the following SQL query to create a new MySQL user:

CREATE USER 'newuser'@'localhost' IDENTIFIED BY 'password';

Replace 'newuser' with the desired username and 'password' with the desired password.

Step 3: Grant Privileges to the New User

Grant appropriate privileges to the new user. For example, to grant all privileges on all databases:

GRANT ALL PRIVILEGES ON *.* TO 'newuser'@'localhost' WITH GRANT OPTION;

Step 4: Flush Privileges

Flush the privileges to apply the changes:

FLUSH PRIVILEGES;

Step 5: Exit MySQL

Exit the MySQL prompt:

exit;
