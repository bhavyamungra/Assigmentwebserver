Step 1: Add PHP Repository

Add the ondrej/php repository to your system by running the following commands:

sudo apt install software-properties-common
sudo add-apt-repository ppa:ondrej/php
sudo apt update

Step 2: Install PHP v8.2

Install PHP v8.2 and its required dependencies:

sudo apt install php8.2 php8.2-cli php8.2-common

Step 3: Verify PHP Installation

After installation, verify that PHP v8.2 is installed correctly by running:

php -v

This command should display the installed PHP version.
