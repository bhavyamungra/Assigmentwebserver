Step 1: Create a PHP Info File

Create a new PHP file with the following content:

<?php
    phpinfo();
    ?>

Save this file as info.php in the document root of your Apache web server (typically /var/www/html/).

Step 2: Access the PHP Info File

Open your web browser and navigate to http://your_server_ip/info.php, replacing your_server_ip with the IP address or domain name of your server.

Step 3: Verify LAMP Stack Components

If your LAMP stack is configured correctly, you should see a page displaying detailed information about your PHP installation.

This page confirms that Apache is working properly, PHP is installed and configured correctly, and your server can serve PHP files.

Step 4: Delete the PHP Info File (Optional)

For security reasons, it's recommended to delete the info.php file after testing your LAMP stack:

sudo rm /var/www/html/info.php
